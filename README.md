# SutoClean

SutoClean is a Python utility that cleans system caches and temporary files.  
**Beta version**: Terminal-only.  
**Version 1.0**: Will feature a graphical user interface (UI) and extra automations.

---

## Project Structure

- **main.py** → Main script that runs the program.  
- **scanner.py** → Responsible for scanning files and folders that can be cleaned.  
- **cleaner.py** → Executes the cleaning of files detected by `scan.py`.
